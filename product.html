Visual Memory Game

Tentang Game

Visual Memory Game adalah sebuah game asah otak berbasis desktop yang dibuat menggunakan library Tkinter dan modul random pada Python. Game ini menantang daya ingat visual pemain melalui kombinasi urutan pola balok warna yang menyala secara acak.

Cara Bermain:

  • Klik tombol START GAME untuk memulai permainan.
  • Perhatikan layar dengan seksama dan ingat urutan balok warna yang menyala.
  • Setelah giliranmu tiba, klik balok-balok warna sesuai dengan urutan persis yang baru saja diperlihatkan.
  • Setiap berhasil melewati sebuah pola, level akan naik dan urutan balok warna yang harus diingat akan menjadi lebih panjang dan menantang!
