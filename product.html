Visual Memory Game

Tentang Game

Game asah otak ini dibuat pakai library Tkinter dan modul random di Python. Game ini bakal nguji daya ingat kamu lewat urutan tombol warna yang menyala secara acak.

Cara Bermain:

  • Klik tombol START GAME buat mulai main.
  • Ingetin baik-baik urutan balok warna yang menyala di layar.
  • Klik lagi balok warna tadi sesuai urutan yang bener.
  • Tiap kali tebakanmu bener, level bakal naik dan urutannya jadi makin panjang.
